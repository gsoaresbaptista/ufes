# Pokemon Battle System

**Atenção:** ENTER = Z

## Como rodar

Copie o repositório para uma pasta local e rode os seguintes comandos:

- `git clone https://github.com/gsoaresbaptista/pokemon.git`
- `cd pokemon/pokemon`
- `npm install`
- `npm start`

E o jogo abrirá no seu navegador padrão.
